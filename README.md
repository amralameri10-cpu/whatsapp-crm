# WhatsApp CRM — نظام إدارة محادثات واتساب

نظام كامل لإدارة محادثات الواتساب مع دعم أدوار متعددة، موافقة على الردود، وذكاء اصطناعي.

---

## ✨ الميزات

- **محادثات الواتساب** — استقبال وإرسال الرسائل + وسائط
- **أدوار ثلاثية**: Super Admin → Admin → Agent مع صلاحيات مخصصة
- **نظام الموافقة** — ردود الموظفين تحتاج موافقة الأدمن (يُفعَّل/يُغلَق لكل موظف أو محادثة)
- **إخفاء البيانات** — رقم العميل مخفي عن الموظف (`***`) حسب الصلاحية
- **Realtime** — الرسائل فورية عبر Pusher
- **AI** — دعم Anthropic Claude + OpenAI GPT + Google Gemini
- **Flow Builder** — أتمتة بسيطة مع node خاص لـ AI
- **كامبينز** — إرسال رسائل جماعية
- **Instances متعددة** — أكثر من رقم واتساب

---

## التثبيت

### 1. تثبيت الحزم
\`\`\`bash
cd whatsapp-crm
npm install
\`\`\`

### 2. إعداد متغيرات البيئة
\`\`\`bash
cp .env.example .env
# عدّل .env بالبيانات الخاصة بك
\`\`\`

### 3. إنشاء قاعدة البيانات
\`\`\`bash
npm run db:setup
\`\`\`

### 4. تشغيل النظام
\`\`\`bash
npm run dev
\`\`\`

---

## أول إعداد

1. اذهب لـ `/login?mode=signup` — أنشئ حساب المسؤول الأول (Super Admin)
2. من `/admin` — أدخل بيانات Evolution API وPusher
3. من إعدادات الموظفين — أضف رقم واتساب عبر QR Code
4. ادعُ الموظفين من صفحة الإعدادات وحدد صلاحية كل واحد

---

## الأدوار

| الدور | يشوف | يقدر |
|-------|------|------|
| Super Admin | كل شيء | إعدادات النظام |
| Admin | كل المحادثات | إدارة الموظفين، الموافقة على الردود |
| Agent | محادثاته المعيّنة فقط | يرد على العملاء |

### صلاحيات Agent قابلة للتخصيص:
- رؤية رقم الهاتف
- الموافقة قبل الإرسال
- استخدام AI
- رؤية كل المحادثات
